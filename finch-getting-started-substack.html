Getting Started with Finch: An AI Coding Assistant That Actually Remembers You

Finch is a coding assistant you run in your terminal. You ask it questions, it reads your code, runs commands, and helps you build things — the same way a senior developer sitting next to you would. Unlike a web chatbot, it can actually open files, search your codebase, run tests, and remember what you worked on last week.

It works with Grok (free with X Premium+), Claude, GPT-4, and several other AI providers. You don’t need to be a developer to use it. You do need a terminal.

This guide will get you from zero to a working Finch installation in about ten minutes.

Step 1: Install iTerm2

Your Mac comes with a built-in terminal app, but iTerm2 is significantly better — it handles the colors, scrolling, and text rendering that Finch uses for its interface.

  1. Go to iterm2.com and click Download
  2. Open the downloaded .zip file (it extracts automatically)
  3. Drag iTerm2 into your Applications folder
  4. Open it from Applications or Spotlight (Cmd+Space, type “iTerm”)

You should see a dark window with a blinking cursor. Leave it open.

Step 2: Get Your API Key

Finch needs an API key to talk to an AI provider. We recommend Grok if you have X Premium+ — it’s included at no extra cost.

Option A: Grok (X Premium+ subscribers)

  1. Go to console.x.ai and sign in with your X account
  2. In the left sidebar, click API Keys
  3. Click Create API Key, give it a name (e.g., “Finch”), click Create
  4. Copy the key — it starts with xai-

Important: The key is shown only once. Save it in your password manager before closing the page.

Option B: Claude (Anthropic)

  1. Go to console.anthropic.com, create an account
  2. Click API Keys → Create Key, copy it

Claude requires a paid account ($5 minimum credit purchase). Usage is billed per request.

Step 3: Download Finch

In iTerm2, paste the command for your Mac:

Apple Silicon (M1, M2, M3, or M4):

curl -L https://github.com/schancel/shammah/releases/latest/download/finch-macos-arm64.tar.gz | tar xz
sudo mv finch /usr/local/bin/

Intel Mac:

curl -L https://github.com/schancel/shammah/releases/latest/download/finch-macos-x86_64.tar.gz | tar xz
sudo mv finch /usr/local/bin/

Not sure which chip you have? Click the Apple menu → About This Mac. If it says “Apple M1/M2/M3/M4”, you have Apple Silicon. If it says “Intel”, use the second command.

After running those commands, macOS will ask for your password. Type it and press Enter — nothing will appear as you type, which is normal.

Verify it worked:

finch --version

Step 4: Run Setup

finch setup

The wizard opens inside your terminal. Use arrow keys to navigate, Enter to select.

  1. On the Teachers tab, press A to add a provider
  2. Select Grok (or Claude)
  3. Paste your API key (Cmd+V to paste in iTerm2)
  4. Press F to finish and save

You only need to do this once.

Step 5: Start Chatting

finch --cloud-only

The --cloud-only flag skips the local AI model download and goes straight to your provider. This is the right choice to get started immediately.

You’ll see a prompt appear. Type your question and press Enter:

> How do I read a file line by line in Python?

Finch will respond, and you can keep the conversation going.

What Finch Can Do

Finch isn’t just a chatbot — it can take actions with your permission.

Read your files:

> Read my README.md and suggest improvements

Search your code:

> Find everywhere I'm using the old API in this project

Run commands:

> Run the tests and tell me what's failing

Remember past sessions:

> What was wrong with the auth bug we fixed last month?

Finch stores every conversation locally and uses semantic search to pull in relevant history. Navigate to your project folder first (cd ~/my-project) and Finch will read and search files right there.

Useful Shortcuts

Command What it does
finch --cloud-only Start immediately, no model download
finch setup Open the setup wizard
/teacher grok Switch to Grok mid-conversation
/teacher claude Switch to Claude mid-conversation
Ctrl+C Cancel a response in progress
Shift+PgUp Scroll up through history

Troubleshooting

macOS won’t open Finch (“unidentified developer”):

xattr -dr com.apple.quarantine /usr/local/bin/finch

“API key invalid”: Re-run finch setup and paste the full key including the xai- prefix.

Finch is slow: If you ran without --cloud-only, it’s downloading a local AI model in the background (one-time). Use --cloud-only while that happens.

Your Privacy

Everything Finch stores is in ~/.finch/ on your computer. No account required. No data sent to us. The only outbound traffic is your message to your AI provider (Grok, Claude, etc.).

To delete all stored memory:

rm ~/.finch/memory.db

Finch is open source and written in Rust. Source and releases at github.com/schancel/shammah. File issues there.
