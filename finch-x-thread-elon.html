If you have X Premium+, you already have free API access to Grok. Here’s how to turn that into a full coding assistant in 10 minutes. 🧵

[2/9] Most “AI coding tools” cost $20–40/month, require a developer account, and forget everything the moment you close the tab.

Finch is different: — runs in your terminal — remembers past sessions — reads and edits your actual files — free with Grok via X Premium+

[3/9] Step 1: Install iTerm2 (better terminal for Mac)

→ iterm2.com → Download → drag to Applications

Then open it. You’ll see a blinking cursor. That’s where we’re working.

[4/9] Step 2: Get your free Grok API key

→ console.x.ai → Sign in with your X account → API Keys → Create API Key → Copy it (starts with xai-)

That’s it. No credit card. Included with Premium+.

[5/9] Step 3: Install Finch

Apple Silicon (M1/M2/M3/M4):

curl -L https://github.com/schancel/shammah/releases/latest/download/finch-macos-arm64.tar.gz | tar xz && sudo mv finch /usr/local/bin/

Intel Mac:

curl -L https://github.com/schancel/shammah/releases/latest/download/finch-macos-x86_64.tar.gz | tar xz && sudo mv finch /usr/local/bin/

Not sure which? Apple menu → About This Mac.

[6/9] Step 4: Configure it

finch setup

Arrow keys to navigate. Press A → pick Grok → paste your key. Press F to save.

30 seconds.

[7/9] Step 5: Use it

finch --cloud-only

Navigate to your project first:

cd ~/my-project
finch --cloud-only

Then ask:

> Read my code and tell me what's broken
> Run the tests and explain what's failing
> What did we change in auth last week?

[8/9] That last one works because Finch stores every conversation locally and searches it across sessions.

Not a chatbot. Not a cloud service. A coding assistant that lives on your machine and actually remembers you.

Everything stays local. No account. No subscription beyond the X Premium+ you already have.
